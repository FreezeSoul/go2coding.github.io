✨ 1: Claude Code Proxy

Claude Code代理与对话监控

Claude Code Proxy 是一个双重用途的解决方案，它既充当 Claude Code 请求至 Anthropic API 的透明代理，实现实时监控与拦截，又能通过一个美观的Web界面可视化并分析用户的Claude API对话。其核心功能包括将所有API交互记录到SQLite数据库，提供可搜索的请求历史、请求/响应体检查和会话串联，同时还提供一个实时仪表盘，用于展示请求与响应，并支持深入的对话分析，包括自动提示词评分、最佳实践评估和响应质量度量。用户只需通过简单的环境变量设置，即可将Claude Code的请求路由至此代理进行全面的监控与分析。

地址：https://github.com/seifghazi/claude-code-proxy

✨ 2: Multi-Modal Researcher

多模态研究播客生成

Multi-Modal Researcher 是一个基于 LangGraph 的自动化工作流项目，旨在利用 Google Gemini 2.5 系列模型的独特能力，实现主题研究和播客生成。用户输入一个研究主题，并可选择提供一个 YouTube 视频 URL，系统便能通过 Gemini 的原生搜索工具进行网络研究，结合对视频的分析（如果提供），将这些见解整合并生成一份带引用的综合报告。此外，它还能进一步生成一份简短的、包含多扬声器（如专家与采访者）对话的播客脚本及其对应的 TTS 音频文件，提供从文本到语音的多模态输出。该项目支持模型选择、内容创作温度以及语音配置等运行时定制。

地址：https://github.com/langchain-ai/multi-modal-researcher

✨ 3: TradingAgents

基于多智能体大语言模型的中文金融交易决策框架

TradingAgents中文增强版是一个基于多智能体大语言模型的金融交易决策框架，它在原有TradingAgents项目基础上，专为中国市场进行了深度本地化和增强。其核心功能在于模拟真实交易公司的专业分工和协同决策流程，通过分析师、研究员、交易员、风险管理和管理层等多类AI智能体的紧密协作，评估市场状况并生成投资建议。该项目的一大亮点是提供了业界最完整的中文文档体系（超过50,000字），并集成了阿里云通义千问等国产大模型，计划支持更多国内LLM，同时支持A股、港股等中国金融市场数据源（如通达信API、Tushare、AkShare）。项目还配备了基于Streamlit的直观中文Web管理界面，支持多级别研究深度配置，实现实时分析进度可视化和结构化结果展示，并通过并行处理和智能缓存等技术优化性能和成本。其关键特性在于前瞻性的AI驱动决策、实战导向的风险管理，以及为中国用户提供无障碍的使用体验，旨在推动AI在中文金融科技领域的应用与发展。

地址：https://github.com/hsliuping/TradingAgents-CN

✨ 4: Rex-Thinker

思维链目标指代模型

Rex-Thinker是一个专注于对象指代任务的创新模型，其核心优势在于引入了链式思考（CoT）推理机制，旨在解决传统方法的不可解释性和无法拒绝不匹配表达的局限性。该项目通过将对象指代问题重构为分阶段的推理任务，首先将指代表达式分解为可解释的子目标（规划），然后逐步评估每个候选对象并以Box Hints进行可视化（行动），最后汇总结果并决定是否给出最终预测或拒绝（总结）。Rex-Thinker基于QwenVL-2.5开发，并采用两阶段训练：首先通过GPT-4o生成的CoT轨迹进行监督微调，随后利用基于GRPO的强化学习进一步提升其推理准确性、泛化能力和拒绝能力。此外，项目还开源了HumanRef-CoT数据集，一个包含高质量分步推理标注的大规模数据集，为模型的CoT训练提供了关键支持，使得Rex-Thinker能够提供忠实、可解释且在域外场景中表现良好的预测。

地址：https://github.com/IDEA-Research/Rex-Thinker

✨ 5: AI小说生成系统

AI小说智能创作系统

“AI小说生成系统”是一个基于LangChain框架的专业AI小说创作工具，其主要功能围绕智能内容生成和项目管理展开。该系统核心支持包括DeepSeek、OpenAI、Claude、Gemini在内的多种大型语言模型，能够实现章节内容的自动化生成。项目关键特性在于其强大的状态管理能力，可以保存并更新小说中主角、物品、关系等章节状态，以及详细的世界设定，确保故事的连贯性和内部逻辑。此外，它集成了智能记忆系统，能有效压缩对话历史，维持长期的上下文理解。为了提供便捷的用户体验，系统配备了交互式的Web界面，并支持多小说项目的独立管理以及生成内容的版本控制与比较，使其成为一个全面、高效的AI辅助小说创作平台。

地址：https://github.com/yefeng2renxing/AIxiezuo
