✨ 1: Deep Agents

深度LLM智能体框架

“Deep Agents"是一个Python包，旨在通过提供一套核心架构，帮助开发者构建能够处理更长、更复杂任务的“深度智能体”。该架构集成了规划工具、子智能体、文件系统访问以及详细的提示词引导机制。其核心功能包括一个高度定制化的系统提示，它极大地提升了智能体处理复杂问题的能力；一个内置的规划工具，用于智能体的内部任务规划；模拟文件系统工具（如ls、edit_file、read_file、write_file），用于在不实际操作本地文件系统的情况下管理状态；以及对子智能体的支持，包括一个通用子智能体和自定义子智能体，以实现“上下文隔离”和专业任务处理。该项目构建于LangGraph之上，允许用户以LangGraph的方式进行灵活交互，并可自定义工具、指令和模型。

地址：https://github.com/hwchase17/deepagents

✨ 2: LangExtract

大模型文本信息结构化提取

LangExtract是一款强大的Python库，旨在利用大型语言模型（LLM）根据用户定义的指令，从非结构化文本中精准提取结构化信息。该项目核心优势在于其能够将每次提取精确溯源到原文位置，确保输出结果的可靠性和结构一致性，并通过优化策略高效处理长文档，提供直观的交互式可视化界面来审阅提取数据。此外，LangExtract支持包括Google Gemini在内的多种云端及本地LLM，并能通过少量示例灵活适应任何领域，无需额外的模型微调。

地址：https://github.com/google/langextract

✨ 3: Claude Code Subagents Collection

Claude Code 专业子代理

Claude Code Subagents Collection是一个为Claude Code设计的综合性AI子代理集合，其核心目标是通过提供36个专业领域专家来显著增强开发工作流。这些子代理覆盖了软件开发生命周期中的广泛领域，包括但不限于后端架构、前端开发、多种编程语言、基础设施与运维、质量保障与安全审计、数据分析与AI/ML工程、以及区块链和加密货币交易等特定专业领域。该项目的主要特点是能够根据任务上下文自动识别并调用最合适的子代理，或允许用户通过自然语言或@提及方式明确指定，从而为复杂的开发挑战提供精准、高效的专业化解决方案，并通过其模块化结构支持定制和多子代理的链式协作，极大地扩展了Claude Code的功能和处理能力。

地址：https://github.com/davepoon/claude-code-subagents-collection

✨ 4: DispatchMail

AI智能收件箱

DispatchMail是一个AI驱动的本地运行电子邮件管理工具，旨在通过人工智能技术自动化收件箱管理。该项目能够实时监控电子邮件，并利用OpenAI的AI代理根据用户自定义的阅读和草稿提示词对邮件内容进行智能处理和分析，同时提供一个本地运行的Web界面，供用户可视化管理邮件、筛选结果、生成邮件草稿和撰写回复。其主要功能包括高度可定制的邮件过滤白名单（支持基于发件人、主题和AI规则）、自动化邮件标签与归档，以及对发件人资料进行深度AI研究。所有邮件数据、配置和用户凭证均存储在本地SQLite数据库中，确保数据隐私，是一个注重本地化部署和用户自定义体验的AI-native邮件处理原型。

地址：https://github.com/dbish/DispatchMail

✨ 5: Claude Sub-Agents Manager

Claude 代码增强 AI 代理

Claude Sub-Agents Manager是一款强大的CLI工具，旨在通过一系列高度专业化的AI子代理，极大地增强Claude Code的开发能力。该项目核心在于为代码审查、自动化测试、智能调试、文档生成以及安全扫描等各类开发任务提供专家级智能助手，并支持从开箱即用到完全自定义的工作流。其关键特性包括与Context-Forge项目的深度集成，能够感知并利用现有PRP（项目需求建议），实现智能上下文管理和并发任务执行。开发者可以通过命令行或直观的Web仪表盘轻松管理和运行这些代理，从而自动化和优化整个软件开发生命周期中的各项任务，并通过钩子系统实现事件驱动的自动化工作流，有效提升开发效率和代码质量。

地址：https://github.com/webdevtodayjason/sub-agents
