✨ 1: Wan2.2

Wan2.2：高效高清视频生成大模型

Wan2.2作为一个开放且先进的大规模视频生成模型，其核心功能和亮点体现在：它创新性地采用了高效的混合专家（MoE）架构，在不增加计算成本的前提下显著提升了模型容量，从而能够生成具备电影级别美学和复杂运动表现的高质量视频。项目中包含专门用于文本到视频（T2V）、图像到视频（I2V）以及高效混合文本-图像到视频（TI2V）的模型，其中TI2V-5B模型尤其突出，它通过高压缩率的Wan2.2-VAE，实现了在消费级GPU上以720P@24fps分辨率快速生成视频，是目前同类模型中的佼佼者。Wan2.2还支持通过AI模型进行智能提示词扩展，以进一步丰富生成视频的细节；并通过FSDP和DeepSpeed Ulysses技术优化了多GPU推理效率。该模型已集成到ComfyUI和Diffusers等主流平台，并在性能对比中展现出超越领先商业模型的卓越能力，全面满足工业界和学术界对先进视频生成技术的需求。

地址：https://github.com/Wan-Video/Wan2.2

✨ 2: GLM-4.5

GLM-4.5智能体混合推理基础模型

GLM-4.5项目提供了一系列专为智能代理设计的基础模型，旨在统一推理、编码和智能代理能力，以满足复杂应用场景的需求。该系列包括参数规模更大的GLM-4.5及其更紧凑高效的GLM-4.5-Air版本，两者皆为混合推理模型，支持复杂的“思考模式”以进行推理和工具使用，同时兼顾快速响应的“非思考模式”。所有模型（包括基础模型和优化过的FP8版本）均已根据MIT开源许可证发布，允许商业应用和二次开发。GLM-4.5在多项行业标准基准测试中展现出卓越性能，排名靠前，并支持工具调用功能，其实现代码已集成到transformers、vLLM和SGLang等主流框架中，可在推荐的硬件配置下进行高效推理和微调，为开发者构建高性能智能代理提供了坚实的基础。

地址：https://github.com/zai-org/GLM-4.5

✨ 3: Automate the Boring Stuff

Claude CLI 自动化命令

“Automate the Boring Stuff”项目下的CCPlugins是一个为Claude Code CLI设计的生产力工具集，旨在通过一系列自动化命令显著减少开发者在重复性任务上的时间投入，每周可节省2-3小时。该项目提供了覆盖开发工作流、代码质量、会话管理以及安全与性能等多个方面的预配置命令，核心功能包括清理项目、自动生成规范化提交信息、格式化代码、自动修复测试、进行全面代码审查、清理不必要注释、修正类型和导入问题、查找待办事项、管理编码会话以及回滚操作。其独特之处在于采用“第一人称”的对话式命令设计，将Claude Code CLI转变为一个智能协作伙伴，而非仅仅是命令执行器，能够根据项目上下文进行智能解读和适应，从而在任何语言和框架下高效提升开发效率和代码质量。

地址：https://github.com/brennercruvinel/CCPlugins

✨ 4: Claude Code Remote

Claude Code 邮箱遥控

Claude Code Remote是一个通过电子邮件实现对Claude Code进行远程控制和管理的工具。其核心在于允许用户在本地启动Claude Code会话后，通过邮件接收任务完成通知，并通过回复邮件直接发送新的命令，从而实现远距离的代码审查、长时间任务监控以及跨地点开发。项目具备邮件通知、邮件指令控制、多行命令支持等主要功能，并通过发件人白名单确保了操作的安全性，使得用户能够便捷地远程管理Claude Code工作流。

地址：https://github.com/JessyTsui/Claude-Code-Remote

✨ 5: Local voice agents on macOS with Pipecat

macOS离线语音AI助手

一个基于Pipecat开源框架的macOS本地实时语音AI代理。此代理的核心功能在于其低延迟（M系列Mac上可达800毫秒以下）的语音到语音交互能力，所有关键组件如语音活动检测、语音转文本、大语言模型处理和文本转语音均在本地运行。项目强调高度的模块化和可定制性，允许用户灵活替换模型（如Silero VAD、MLX Whisper、Gemma3 12B和Kokoro TTS）或重新配置整个处理管道，并支持工具调用、并行推理等高级功能。代理与基于React的Web客户端之间通过低延迟、无服务器的WebRTC连接进行通信，确保了高效的实时音频传输。为了支持大语言模型，用户需要运行一个本地的OpenAI兼容LLM服务器。

地址：https://github.com/kwindla/macos-local-voice-agents
