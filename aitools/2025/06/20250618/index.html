✨ 1: RAG-Anything

RAG-Anything：一体化多模态文档处理RAG系统

RAG-Anything 是一个基于 LightRAG 构建的 All-in-One 多模态文档处理 RAG 系统，旨在解决传统 RAG 系统无法有效处理包含文本、图像、表格、公式、图表和多媒体等多样化内容的问题。它的主要功能、核心要点和关键特性如下：

  1. 端到端多模态管道：提供从文档摄取和解析到智能多模态查询应答的完整工作流程。
  2. 通用文档支持：无缝处理 PDF、Office 文档（DOC/DOCX/PPT/PPTX/XLS/XLSX）、图像和各种文件格式。
  3. 专门的内容分析：为图像、表格、数学公式和异构内容类型提供专用处理器。
  4. 多模态知识图谱：自动提取实体和跨模态关系，以增强理解。
  5. 自适应处理模式：灵活的基于 MinerU 的解析或直接多模态内容注入工作流程。
  6. 混合智能检索：高级搜索功能，跨越文本和多模态内容，具有上下文理解能力。

地址：https://github.com/HKUDS/RAG-Anything

✨ 2: secure-rules-files

AI生成代码安全基线规则文件

secure-rules-files 提供了一套 AI 原生的基线规则文件，旨在提高生成代码的安全性。这些规则文件通过编码常见编程语言和框架的安全最佳实践，为代码生成过程提供安全指导。这些规则本身也是通过 AI 生成的，并针对多个流行的 AI 编码助手和工具（如 Cursor, Cline, Claude, Windsurf, Codex, Aider, GitHub Copilot）进行了适配。该项目允许用户贡献新的语言和框架，或者改进用于生成规则文件的 prompt。

地址：https://github.com/wiz-sec-public/secure-rules-files

✨ 3: DeepResearch Bench

深度研究代理综合评测基准（DeepResearch Bench）

DeepResearch Bench，旨在为深度研究代理（DRAs）提供一个全面的评估基准。

  1. 综合性基准： 提供100个博士级别的研究任务，涵盖22个不同的领域，包括科学技术、金融商业、软件以及其他领域。这些任务经过领域专家精心设计，旨在评估DRAs的深度研究能力。

  2. 真实需求分析： 通过分析来自实际用户的大量匿名网络搜索查询，确保基准任务与现实世界的研究需求保持一致。

  3. 专家任务收集： 由博士级别的专家和高级从业人员设计任务，并经过严格的人工筛选，保证任务的质量、清晰度、真实性和挑战性。

  4. 双重评估方法： 引入两种互补的评估方法，全面评估深度研究代理：

    • RACE（Reference-based Adaptive Criteria-driven Evaluation）： 基于参考答案的自适应标准驱动评估，通过动态生成的任务特定评估标准，评估报告的生成质量，关注报告的全面性、深度、指令遵循和可读性。
    • FACT（Framework for Factual Abundance and Citation Trustworthiness）： 用于评估信息检索和可信度，通过提取报告中的事实性陈述和引用的来源，并验证这些来源是否支持陈述，来评估模型的信息检索和依据能力。

  5. 性能评估结果： 提供不同模型架构和方法在RACE和FACT评估指标上的详细性能比较，揭示了深度研究代理和传统LLM的性能差异。

  6. 易于使用的评估框架： 提供清晰的项目结构、安装指南和快速开始流程，方便用户使用自己的模型进行评估。同时，支持自定义LLM接口，方便集成不同的模型。

地址：https://github.com/Ayanami0730/deep_research_bench

✨ 4: Office-PowerPoint-MCP-Server

PowerPoint MCP Server: Python演示文稿操作工具

Office-PowerPoint-MCP-Server 是一个使用 MCP (Model Context Protocol) 协议，通过 python-pptx 库操作 PowerPoint 演示文稿的服务器。它主要功能包括：创建、编辑和操作 PowerPoint 演示文稿，能支持添加幻灯片、填充文本占位符、添加图片、添加文本框、添加表格、添加自选图形以及添加和操作各种图表。该项目可以通过 Smithery 自动安装，或者手动安装，并提供了使用 UVX 的配置选项。服务器提供了一系列工具用于演示文稿、幻灯片、文本、图像、表格、形状和图表的操作，并提供了详细的使用示例。

地址：https://github.com/GongRzhe/Office-PowerPoint-MCP-Server

✨ 5: Django MCP Server

Django MCP Server: 连接 AI 智能体与 Django 应用

Django MCP Server 是一个 Django 扩展，实现了模型上下文协议 (MCP)，使得 MCP 客户端和 AI 代理能够与任何 Django 应用程序无缝交互。通过简单的配置和声明式工具，该项目旨在简化 Django 模型和逻辑的暴露，方便 AI 代理进行查询和交互。此外，它还支持将 Django REST Framework API 转化为 MCP 工具，并且与多种 MCP 客户端（例如 Google Agent Development Kit 和 Claude AI）兼容。总而言之，该项目的主要目标是弥合 Django 应用和 AI 代理之间的差距，提供一个安全高效的集成方案。

地址：https://github.com/omarbenhamid/django-mcp-server
