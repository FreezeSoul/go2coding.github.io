✨ 1: watercrawl

WaterCrawl: 强大的自托管网络爬虫与数据提取工具

WaterCrawl是一个基于Python、Django、Scrapy和Celery构建的Web应用程序，主要用于网页抓取和数据提取。它具有高度可定制的抓取选项，支持多语言搜索和抓取，并提供强大的搜索引擎和REST API。该项目还具有异步处理能力，丰富的生态系统，支持自托管和开源，并提供高级结果处理功能。此外，它还提供多种语言的客户端SDK以及与Dify和N8N等AI/自动化平台的集成。

地址：https://github.com/watercrawl/watercrawl

✨ 2: graph-rag-agent

GraphRAG与DeepSearch融合的智能问答Agent

graph-rag-agent，主要功能是结合 GraphRAG 和 私域 Deep Search 来构建一个可解释、可推理的智能问答系统。它融合了多 Agent 协作和知识图谱增强，旨在提供一个完整的 RAG 智能交互解决方案。以下是其核心功能和关键特性：

  1. GraphRAG 实现：项目从零开始完整实现了 GraphRAG 的核心功能，将知识表示为图结构，从而能够进行关系推理。
  2. DeepSearch 创新融合：创新性地将 DeepSearch 框架（通常基于向量数据库）与知识图谱相结合，利用图结构进行增强。
  3. 多 Agent 协同架构：实现了不同类型的 Agent 协同工作，提升了处理复杂问题的能力。
  4. 完整的评估系统：提供 20+ 种评估指标，全面衡量系统性能。
  5. 增量更新机制：支持知识图谱的动态增量构建与智能去重。
  6. 思考过程可视化：展示 AI 的推理轨迹，提高可解释性和透明度。

地址：https://github.com/1517005260/graph-rag-agent

✨ 3: Claudia

Claudia：Claude Code的强大GUI工具

Claudia 是一款强大的桌面应用程序，旨在提升用户与 Claude Code 的交互体验。它通过图形用户界面(GUI)简化了 Claude Code 会话的管理、自定义代理的创建、使用情况的跟踪等任务。

主要功能和特点：

  • 项目与会话管理： 提供可视化的项目浏览器，方便用户浏览和管理 Claude Code 项目；支持会话历史记录，方便用户查看和恢复过去的编码会话；内置智能搜索，快速查找项目和会话；提供会话洞察，一览会话的初始信息、时间戳和元数据。

  • CC 代理： 允许创建具有自定义系统提示和行为的专用代理；建立代理库，用于存储不同任务的专用代理；支持在安全沙箱环境中运行代理，具有细粒度的权限控制；详细记录代理运行历史，包括日志和性能指标。

  • 高级沙箱： 提供操作系统级别的安全沙箱(Linux 上的 seccomp、macOS 上的 Seatbelt)；支持创建具有精细访问控制的可重用安全配置文件；实时监控和记录所有安全违规行为；支持跨团队和系统共享沙箱配置文件。

  • 使用分析仪表盘： 实时监控 Claude API 的使用情况和成本；详细分析模型、项目和时间段的 Token 使用情况；通过可视化图表展示使用趋势和模式；支持导出使用数据，用于统计和分析。

  • MCP 服务器管理： 从中央用户界面管理模型上下文协议(MCP)服务器；通过用户界面或导入现有配置轻松添加服务器；在使用前验证服务器连接；从 Claude Desktop 导入服务器配置。

  • 时间线与检查点： 在编码会话中的任何时间点创建检查点；使用分支时间线浏览会话历史记录；一键恢复到任何检查点；从现有检查点创建新分支；查看检查点之间的具体更改。

  • CLAUDE.md 管理： 在应用程序内直接编辑 CLAUDE.md 文件；实时预览 Markdown 渲染效果；扫描项目中的所有 CLAUDE.md 文件；提供完整的 Markdown 语法高亮支持。

地址：https://github.com/getAsterisk/claudia

✨ 4: cpdown

网页转Markdown：cpdown浏览器扩展

cpdown 是一款浏览器扩展程序，其主要功能是将任何网页的内容复制为整洁、格式化的 Markdown 文本。它提供一键复制网页内容和 YouTube 字幕的功能，使用 Defuddle 或 Mozilla 的 Readability 提取主要内容，并移除不必要的 HTML 元素。此外，该扩展还能显示复制内容的 token 数量，并支持键盘快捷键。用户可以通过设置选项进行个性化配置，例如选择使用 Defuddle 或 Readability，以及是否显示成功提示等。

地址：https://github.com/ysm-dev/cpdown

✨ 5: Agents Towards Production

GenAI智能体生产实战手册

“Agents Towards Production”，旨在提供一套开源工具、模式和代码示例，帮助开发者将 AI Agents 从零开始构建并扩展到生产环境。它提供了一系列可运行的教程，涵盖了 GenAI Agent 栈的各个构建模块，包括编排、内存、可观测性、部署、安全性等。该项目旨在为开发者提供实际操作的指导，使其能够在几分钟内将概念转化为可运行的 Agent。它还提供订阅更新的途径，包括 Newsletter，以帮助用户获取最新的前沿见解和免费教程。简而言之，该项目是一个全面的资源库，旨在简化 AI Agent 的生产部署过程，并提供涵盖整个生命周期的解决方案。

地址：https://github.com/NirDiamant/agents-towards-production
