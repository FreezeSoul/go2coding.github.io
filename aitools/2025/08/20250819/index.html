✨ 1: mobile-use

手机自然语言自动化AI代理

mobile-use是一个强大的开源AI代理，旨在通过自然语言指令自动化控制Android和iOS设备。其核心功能在于智能识别并导航应用界面，能够响应用户命令执行复杂的手机任务，并支持从任意应用中提取数据并按指定格式（如JSON）进行结构化。该项目具有高度的可扩展性和定制性，允许用户集成不同的LLM模型，并在AndroidWorld基准测试中取得了卓越的性能。用户只需简单地用自然语言描述操作或数据提取需求，该工具便能高效地完成移动设备上的交互。

地址：https://github.com/minitap-ai/mobile-use

✨ 2: Claude Code Spec-Driven Development

Claude 智能规格驱动开发

Claude Code Spec-Driven Development是一个基于Claude Code斜杠命令和CLAUDE.md的项目，旨在系统化并高效地实施规格驱动开发（Spec-Driven Development, SDD）。它通过提供一系列集成命令，自动化了开发流程中的关键文档生成，包括项目引导文档（如产品、技术、架构规范）、功能需求、技术设计和具体实施任务。该项目核心在于强制执行严格的多阶段人工评审与批准工作流，确保每个开发阶段的质量与一致性，同时利用自动化机制实现任务进度追踪、规格合规性检查，并在整个开发周期中保持项目上下文和防止方向性偏差，从而提升开发效率和产品质量。

地址：https://github.com/gotalab/claude-code-spec

✨ 3: Thyme

Thyme：代码执行式高阶视觉推理

Thyme 是一个开创性的多模态大语言模型，其核心在于通过自主生成和执行多样化的图像处理与计算操作的代码，从而超越了传统的“以图思考”范式。该项目显著提升了模型在高分辨率感知和复杂推理任务上的性能，其实现得益于创新的两阶段训练策略，即监督微调（SFT）与强化学习（RL）的结合，并由独特的 GRPO-ATS 算法赋能，以在推理探索和代码执行精度之间取得精妙平衡。它提供了完整的数据准备指南、详细的SFT和RL训练流程，支持通过 VLMEvalKit 进行基准评估，并提供便捷的推理示例，清晰展示了其根据用户问题生成并运行代码（如图像裁剪、放大）以辅助解决视觉难题的能力。

地址：https://github.com/yfzhang114/Thyme

✨ 4: GeminiCLI to API

Gemini 转 OpenAI 兼容 API

“GeminiCLI to API"项目旨在将Gemini命令行接口转换为OpenAI兼容的API，以解决Gemini API频繁中断和质量下降的问题。其核心功能在于提供标准的OpenAI API端点（/v1/chat/completions和/v1/models），支持实时流式响应和伪流式回退机制，并具备智能凭证管理能力，通过多个Google OAuth凭证实现自动轮换、冗余认证增强稳定性以及负载均衡与并发请求支持，同时还提供简化的Web认证界面。该项目支持具有1M上下文窗口和1500次请求额度的gemini-2.5-pro系列模型。此工具仅供个人学习、研究和非商业用途，任何形式的商业使用均被严格禁止，使用者需自行承担由此产生的一切法律责任；在云服务器或其他远程环境部署时，需注意OAuth认证流程仅支持本地访问，因此需要先在本地完成认证并获取凭证文件再上传使用。

地址：https://github.com/su-kaka/gcli2api

✨ 5: DeepRolePlay

防遗忘智能角色扮演系统

DeepRolePlay是一个旨在解决大型语言模型在角色扮演中“角色遗忘”和“剧情断裂”问题的深度角色扮演系统。它通过自动化记忆管理、智能场景更新以及结构化的数据管理系统（包括记忆闪回处理、场景更新管理和表格管理）来确保AI角色设定和剧情的持久连贯性，有效降低了长对话的Token消耗。该系统还集成了智能联网功能以丰富背景信息，提供直接在聊天界面中操作的后端命令，并支持基于对话内容智能异步生成图像，从而全面提升了角色扮演的沉浸感和一致性。其设计高度兼容任何OpenAI风格的API，允许用户灵活配置不同模型以优化性能和成本。

地址：https://github.com/howyoungchen/deepRolePlay
