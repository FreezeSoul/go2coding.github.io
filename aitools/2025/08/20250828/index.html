✨ 1: ChinarXiv

Arxiv论文智能翻译PDF生成

ChinarXiv 是一个将 GPT Academic 项目中的 Arxiv 论文翻译功能独立出来的工具，致力于提供更便捷的 Arxiv 论文翻译部署和集成服务。该项目实现了从 Arxiv 链接获取论文到生成翻译后 PDF 的完整流程，其核心特点包括基于 LaTeX 结构的智能内容切分以保持文档完整性，支持多线程并行翻译以提高效率，并利用缓存机制避免重复工作。此外，它内置了强大的 LaTeX 编译服务，可处理依赖文件并支持中文排版，同时允许用户通过自定义术语词典和翻译要求来确保译文的学术一致性和专业性，并通过智能错误处理与恢复机制保障翻译的稳定性。用户可以灵活配置 API 接口、LLM 模型及并发参数，以适应不同的翻译需求，并且可以方便地通过 Docker 部署 LaTeX 编译服务。

地址：https://github.com/kaixindelele/chinarxiv

✨ 2: HackGPT Enterprise

企业级AI渗透测试平台

HackGPT Enterprise是一款专为企业安全团队打造的生产级、云原生AI驱动渗透测试平台。它集成先进的AI引擎，支持GPT-4、本地大模型及机器学习能力，可实现模式识别、零日漏洞检测、风险智能分析和自动化生成符合OWASP、NIST等多项国际合规标准的详细报告。该平台采用微服务架构，基于Docker和Kubernetes实现高可用性、性能与可扩展性，并支持AWS、Azure、GCP等主流多云部署。其核心安全特性包括基于RBAC和LDAP/AD集成的企业级认证授权、AES-256-GCM数据加密和全面的审计日志。HackGPT Enterprise提供一套增强的六阶段渗透测试方法论，通过AI驱动自动化提升各阶段效率，并通过命令行接口、REST API、可视化Web仪表盘及语音命令提供灵活的交互方式。此外，它还整合了Prometheus、Grafana和ELK栈，实现了全面的实时监控、告警和深度分析能力，同时允许高度定制AI模型和合规框架以适应特定企业需求。

地址：https://github.com/yashab-cyber/HackGpt

✨ 3: ASTRA

AI软件助手自主红队系统

ASTRA是一个为AI软件助手设计的自主时空红队系统，它在亚马逊Nova AI挑战赛中荣获攻击团队冠军，成功率高达90%以上。该系统通过结构化领域建模，无需预定义基准即可自动生成高质量的违规提示和测试用例；它采用多轮对话框架，通过对输入空间（空间）和推理过程（时间）进行自适应探测来识别并利用目标系统的漏洞；更重要的是，ASTRA具备自演进能力，能记录成功案例并调整策略以提升漏洞发现的成功率，实现全生命周期的自动化安全评估，且整个过程无需人工干预。

地址：https://github.com/PurCL/ASTRA

✨ 4: PRISM-INSIGHT

AI股票分析与模拟交易

PRISM-INSIGHT 是一个基于 AI 的综合性股票分析和交易模拟系统，其核心功能在于利用 GPT-4.1 多智能体系统进行专家级的股票分析，能自动捕捉（上午/下午）市场中的飙升股，并实时通过 Telegram 推送分析报告。该项目还集成了基于 GPT-5 的交易策略模拟功能，已显示出251.39%的累计收益率，并支持股价、交易量、总市值等多种可视化图表生成。此外，它通过 Anthropic Claude 提供 Telegram 机器人对话功能，并利用多个 MCP 服务器（如kospi_kosdaq、firecrawl、perplexity、sqlite）获取并存储数据，允许用户高度定制飙升股捕捉标准、AI 提示词和图表样式，旨在为用户提供专业的市场洞察和策略建议，但强调分析信息仅供参考，不构成投资决策。

地址：https://github.com/dragon1086/prism-insight

✨ 5: Qoder-Free

Qoder身份指纹重置工具

Qoder-Free 是一款现代化的跨平台图形用户界面工具，主要功能是全面重置 Qoder 应用程序的用户身份信息和设备指纹，以有效规避重复注册检测。它提供一系列深度清理能力，包括生成全新的机器ID和遥测数据、执行最彻底的硬件指纹重置（能根据操作系统类型智能生成虚假硬件配置以干扰检测），以及深度清除网络状态、信任令牌、本地存储、系统级身份文件和崩溃报告等所有隐藏的身份痕迹，并特设登录身份清理功能。该工具采用 PyQt5 构建，提供美观的多语言界面，支持实时日志显示、自动状态检测和操作前确认等安全特性，并集成“一键完整重置”功能，集所有增强型清理于一体，实现最全面的反检测效果，同时允许用户智能选择保留或彻底清除对话记录。

地址：https://github.com/itandelin/qoder-free
